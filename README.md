# IP_ADDRESS
# Пример приложения для получения IP-адреса с использованием Windows Forms
![Снимок экрана 2023-07-11 020237](https://github.com/NurzhanAuelbek/IP_ADDRESS/assets/82052572/82589467-e360-444d-87a4-e2dac7771a32)

Добро пожаловать в репозиторий примера приложения для получения IP-адреса, построенного с использованием Windows Forms!

## О проекте

Этот проект представляет собой простое приложение, которое отображает имя хоста и IP-адрес локальной машины. Оно использует Windows Forms для создания графического интерфейса.

## Функциональность приложения

- Получение имени хоста.
- Получение IP-адреса локальной машины.

## Начало работы

Чтобы запустить приложение и получить IP-адрес, выполните следующие шаги:

1. Клонируйте репозиторий на свое локальное устройство с помощью следующей команды: git clone https://github.com/NurzhanAuelbek/IP_ADDRESS.git

2. Откройте проект в вашей интегрированной среде разработки (IDE), например, Visual Studio.

3. Соберите и запустите проект. Приложение отобразит имя хоста и IP-адрес локальной машины в текстовых полях.
